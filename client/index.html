Dance Teaching Aid


This interactive website is designed to help teachers create a ballroom dance routine in a more efficient manner than is possible with the existing recourses. Once you finish making a routine you will be able to see all the technical details in tabular form. This website's main focus is on syllabus figures, but there's also an option for account holders to add their own figures into the mix! Happy dance-making!
